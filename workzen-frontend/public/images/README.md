# Logo Files

Please add the following images to this folder:

## Required Files

1. **logo.png** - Main circular logo (512x512px recommended)
   - This is your circular WorkZen logo
   - Used in: Sidebar, Login page, Settings page
   - Format: PNG with transparency
   - Recommended size: 512x512px or higher

2. **favicon.ico** - Browser tab icon
   - Convert your logo to .ico format
   - Recommended sizes: 16x16, 32x32, 48x48
   - You can use online tools like favicon.io to convert your logo

## How to Add Your Logo

1. **Crop your circular logo:**
   - Use an image editor (Photoshop, GIMP, Canva, etc.)
   - Crop to include just the circular part
   - Export as PNG with transparent background
   - Save as `logo.png` in this folder

2. **Create favicon:**
   - Use your logo.png
   - Visit https://favicon.io/favicon-converter/
   - Upload your logo and download the generated favicon.ico
   - Save as `favicon.ico` in this folder
